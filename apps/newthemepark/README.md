# NewThemePark 風ミニ遊園地経営ゲーム

`build 2026-02-10-newthemepark-v2` の1ファイル完結ゲームです。  
Canvas 描画で、道づくり・施設配置・来園者の移動（BFS経路探索）・簡易経営を遊べます。

## 起動方法

### 1) Chromeで直接開く
1. `apps/newthemepark/index.html` を Chrome にドラッグ＆ドロップ
2. そのままプレイ

### 2) 問題が出る場合（推奨）
`file://` 直開きだと環境によって localStorage の挙動が不安定なことがあります。  
以下の簡易サーバー起動を使ってください。

```bash
py -m http.server 8000
```

> `py` が無い場合は `python -m http.server 8000` でもOKです。

ブラウザで次を開く：

- <http://localhost:8000/apps/newthemepark/>

## 操作方法（キー一覧）

- 建設ツール
  - `1`: 道
  - `2`: アトラクション
  - `3`: 売店
  - `4`: 入口ゲート（1つのみ、置き直しで移動）
  - `X`: 消しゴム
- カメラ
  - `WASD` / `↑↓←→`: スクロール移動
  - 右ドラッグ: スクロール移動
  - マウスホイール: ズーム
- 時間操作
  - `Space`: 一時停止/再開
  - `Q` / `E`: 速度切替（0.5x / 1x / 2x / 4x）
- 見るモード
  - `I`: inspect（見る）モード切替
  - `Esc`: パネルを閉じる / 選択解除
- セーブ関連
  - `S`: セーブ（localStorage）
  - `L`: ロード（localStorage）
  - `N`: ニューゲーム
- 建設
  - 左クリックで配置/削除
  - 配置プレビュー：緑の半透明=設置可能、赤=設置不可
- 演出
  - 収益発生時に施設上へ `+¥` のフロート表示

## 目的（どう遊ぶと楽しいか）

道をつないで、道に隣接する草地へアトラクションや売店を増やし、来園者を回遊させて収益を伸ばすのが目的です。  
日次維持費でお金は減るので、入口と道の導線を整えつつ、黒字運営を目指してください。

## ルールの要点

- 施設（アトラクション/売店）は「道に隣接した草地」にのみ配置可能
- 資金不足時は建設不可（HUDに理由表示）
- 入口ゲートが無い間は来園者がスポーンしない
- 来園者は入口→道→施設→収益発生→再行動/退場 の流れで行動
