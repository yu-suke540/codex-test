# 哲学セッション映像作品ジェネレーター

このフォルダには、1ファイル完結のブラウザアプリ「哲学セッション映像作品ジェネレーター」が入っています。Chromeで`index.html`を開くだけで動作します。

## できること

- 哲学的な問いをランダムまたは手動で選択
- メモ入力に反応する抽象映像（Canvas）
- BGMプリセット（雨/波/森/宇宙/低音ドローン）と音量調整付きの環境音（WebAudio）
- Canvas + 音の録画（MediaRecorderでWebMダウンロード）
- セッションの保存（localStorage）
- メモのMarkdown/JSONエクスポート

## 使い方

1. `apps/philo-cinema/index.html` をChromeで開きます。
2. 問いを選択し、メモを入力します。
3. 必要に応じてBGMプリセットと音量を調整し、「環境音を開始」を押して音を再生します。
4. 「録画開始」→「録画停止」でWebM動画をダウンロードできます。
5. 「セッションを保存」を押すと、問い・メモ・日時が保存されます。
6. メモはMarkdown/JSONでダウンロードできます。

## 注意点

- 音はブラウザの仕様上、必ずユーザー操作後に開始されます。
- 録画はMediaRecorderが使えるブラウザで動作します。
- 保存データはブラウザのlocalStorageに保存されます。
