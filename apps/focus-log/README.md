# 集中タイマー付き作業ログ

集中時間を計測しながら作業ログを残す、ブラウザだけで動くシンプルなアプリです。データは`localStorage`に保存され、再読み込みしても残ります。

## 使い方

1. `apps/focus-log/index.html` をブラウザで開きます。
2. 「開始」を押してタイマーをスタートします。
3. 作業が終わったら「停止」を押します。
4. 必要に応じてカテゴリやメモを入力し、「ログを保存」を押します。
5. 一覧に記録が追加され、今日の合計・カテゴリ別合計が更新されます。

## 主な機能

- タイマーの開始／停止／リセット
- 作業カテゴリ（任意）とメモ（任意）の保存
- 記録内容：開始時刻、終了時刻、所要時間（分）、カテゴリ、メモ、記録日時
- 一覧表示（新しい順）と各行の削除
- 今日の合計時間とカテゴリ別合計（今日）の集計表示
- `localStorage` に保存して永続化
- CSVエクスポート（一覧の全件）

## 注意点

- データはブラウザの`localStorage`に保存されるため、同じブラウザ／同じ端末でのみ引き継がれます。
- 「全件削除」は確認ダイアログが表示され、OKを選ぶと全ての記録が削除されます。
