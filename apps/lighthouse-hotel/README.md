# 灯台ホテルの来客名簿

ブラウザだけで遊べる短編ノベルゲームです。嵐の夜に灯台ホテルへ到着したあなたは、名簿と鍵の違和感から真実へ近づきます。選択肢によって結末が分岐します。

## 遊び方
1. `index.html` をChromeなどのブラウザで開きます。
2. 画面中央の物語本文を読み、表示される選択肢ボタンをクリックして進めます。
3. 進行状況（章/シーン番号）とバックログで、進んだ内容を確認できます。

## セーブ / ロード
- **セーブ**: 「セーブ」ボタンを押すと、現在の `sceneId` とプレイヤー状態（フラグ）が `localStorage` に保存されます。
- **ロード**: 「ロード」ボタンを押すと、保存されているデータから再開します。
- **最初から**: 「最初から」ボタンでタイトル直後のシーンに戻ります。

## エンディング
3種類の結末があります。選択肢と行動によって到達するエンディングが変化します。
