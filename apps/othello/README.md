# 白黒オセロ（リバーシ）

## 遊び方
- `index.html` をChromeで開くと、8×8のオセロが始まります。
- 黒番から開始します。
- 置けるマスだけがハイライトされ、クリック可能になります。
- 置いた後は自動で8方向の石をひっくり返します。
- 手番とスコア（黒・白の枚数）が画面上部に表示されます。
- 置ける手がない場合は自動でパスし、画面に通知が出ます。
- 両者とも置ける手がない場合はゲーム終了となり、勝敗が表示されます。
- 「リスタート」ボタンで初期配置からやり直せます。

## 画面表示
- 画面上部に `build: 2026-02-09-othello-v1` が常に表示されます。
- 同じ文字列がブラウザのタイトルにも入っています。
