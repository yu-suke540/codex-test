# Theme Park Prototype（ステップ2）

このディレクトリは、ブラウザで動く 2D 遊園地経営ゲームの土台です。
ステップ2では、草地タイル描画に加えて、ホバー座標表示と道タイルのトグル配置を実装しています。

## 起動方法

### 1. Chrome で直接開く
1. `apps/themepark/index.html` を Chrome で開きます。

### 2. 簡易サーバで開く（直接開いて問題がある場合）
1. リポジトリのルートで次を実行します。
   - `py -m http.server 8000`
   - または `python -m http.server 8000`
2. Chrome で `http://localhost:8000/apps/themepark/index.html` を開きます。

## ステップ2の操作説明

- マウスを動かすと、HUD の `hover: (x, y)` が現在のタイル座標に更新されます。
- 左クリックで道タイルを置きます。
- 同じタイルをもう一度左クリックすると道タイルを消します（トグル）。
- `road count` に現在の道タイル総数が表示されます。

## ステップ2の確認ポイント

- ホバー中のタイル座標表示がマウス移動に合わせて変化する。
- クリックで道タイルを置ける/消せる。
- ウィンドウをリサイズしても、配置済みの道タイルが残る。
- ブラウザの開発者ツール Console にエラーが出ていない。
