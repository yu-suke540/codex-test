# Theme Park Prototype（ステップ3）

このディレクトリは、ブラウザで動く 2D 遊園地経営ゲームの土台です。
ステップ3では、道に加えて施設（アトラクション）配置とツール切替を実装しています。

## 起動方法

### 1. Chrome で直接開く
1. `apps/themepark/index.html` を Chrome で開きます。

### 2. 簡易サーバで開く（直接開いて問題がある場合）
1. リポジトリのルートで次を実行します。
   - `py -m http.server 8000`
   - または `python -m http.server 8000`
2. Chrome で `http://localhost:8000/apps/themepark/index.html` を開きます。

## ステップ3の操作方法

- ツール切替:
  - `R` キー: 道ツール（road）
  - `A` キー: 施設ツール（attraction）
  - `X` キー: 消しゴムツール（erase）
- クリック挙動:
  - 道ツール: クリックで道タイルを配置/削除（トグル）
  - 施設ツール: クリックで施設を配置（既存施設がある場所は変更なし）
    - 施設は「上下左右いずれかが道に接している草地タイル」にのみ配置可能
    - 条件を満たさない場合は HUD にメッセージ表示
  - 消しゴムツール: クリックしたタイルの道または施設を削除（施設優先）

## ステップ3の確認ポイント

- `R/A/X` でツール切替ができる。
- 道ツールで道の配置/削除ができる。
- 施設が道の隣にだけ置ける。
- 消しゴムで道/施設を消せる。
- ウィンドウをリサイズしても、配置済みの道・施設が残る。
- ブラウザの開発者ツール Console にエラーが出ていない。
